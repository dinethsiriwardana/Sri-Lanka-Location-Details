About

SL Cities

This application provides information about cities, districts, and provinces in Sri Lanka.

You can explore location data through the dashboard, including:

  • Summary statistics of provinces, districts, and cities
  • Search and filter cities by various criteria
  • View detailed information about provinces and districts
API

This application is built on top of an API that provides access to Sri Lankan geographical data.

The API is documented using OpenAPI specification and provides endpoints to retrieve information about cities, districts, and provinces.

For more information about the API, check out the API documentation.

API Endpoints
Base URL
  • Production: https://slcities.live/api
  • Local: http://localhost:3000/api
Cities
  • GET /cities - Retrieve all cities
  • GET /cities/district/{districtName} - Retrieve cities by district
  • GET /cities/province/{provinceName} - Retrieve cities by province
  • GET /cities/postcode/{postcode} - Retrieve a city by postcode
  • GET /cities/search?q={searchTerm}&lang={language} - Search cities
  • GET /cities/nearby?lat={latitude}&lon={longitude}&radius={radius} - Find nearby cities
Provinces & Districts
  • GET /provinces - Retrieve all provinces
  • GET /districts - Retrieve all districts
  • GET /districts/province/{provinceId} - Retrieve districts by province ID
  • GET /districts/province-name/{provinceName}?lang={language} - Retrieve districts by province name
  • GET /summary - Get a summary of provinces with district and city counts
